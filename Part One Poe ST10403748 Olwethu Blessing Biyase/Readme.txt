# Part One Poe
Recipe Program - This is a console program that allows the user to enter the details of a recipe, and then view the recipe in a neat format, scale the recipe, reset the quantities, reset the recipe, or exit the program.

Requirements 
Microsoft .NET Framework 4.5 or later
Microsoft Visual Studio or another C# compiler

How to Compile
Clone or download the repository.
Open the solution file (RecipeProgram.sln) in Visual Studio.
Press F6 or select "Build Solution" from the "Build" menu.

How to Run
Open the command prompt or terminal.
Navigate to the directory containing the compiled program (RecipeProgram.exe).
Run the program by typing RecipeProgram.exe and pressing Enter.
Follow the prompts to enter the details of the recipe and choose options.


How to Use
Enter the number of ingredients.
For each ingredient, enter the name, quantity, and unit of measurement.
Enter the number of steps.
For each step, enter a description of what the user should do.
Choose an option : Display recipe: Displays the full recipe, including the ingredients and steps, in a neat format to the user.
Scale recipe: Allows the user to scale the recipe by a factor of 0.5 (half), 2 (double), or 3 (triple). All the ingredient quantities are changed accordingly when the recipe is displayed.
Reset quantities: Resets all the ingredient quantities to their original values.
Reset recipe: Clears all the data to enter a new recipe.
Exit: Exits the program.
